Premium Server Perks

All settings marked with a can be changed in your server's dashboard. These perks are:

Resend

Whenever someone curses, it send the users message again with curse words blocked out using spoilers, hashtags, or stars. Creating the illusion of a message that has the curse removed.

Multi-Line

This combines a line of messages from the same user to create one full message, this removes the ability to bypass the bot using multiple messages.

Font Recognition

Heavily maintained and accurate font recognition, removing a users ability to bypass curses using character lookalikes.

Ignore Channels

Gives you the ability to set a list of channels to completely ignore curses from, rather than requiring you to set them to NSFW.

Increased Maximums

A list of increases:

  • Filter: Maximum words from 150 to 1500
  • Uncensor List: Maximum words from 150 to 1500
  • Pop Message: Delete after time from 120 seconds to 600
  • Pop message: Max content size from 200 characters to 1000

Less noteable:

  • DM Command Responses: Makes responses to commands respond in the invokers DM's rather than in the same channel.
  • Long lasting HelpME codes, allowing staff to diagnose your problem over the span of months.
