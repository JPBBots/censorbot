Premium Server Perks

All settings marked with a can be changed in your server's dashboard. These perks are:

Resend

Whenever someone curses, it send the users message again with curse words blocked out using spoilers, hashtags, or stars. Creating the illusion of a message that has the curse removed.

Multi-Line

This combines a line of messages from the same user to create one full message, this removes the ability to bypass the bot using multiple messages.

Anti-NSFW Images

Discords built in anti-nsfw images leaves some to be desired, but we've added a custom AI filter which runs it through another check that's more stringent!

Remove Roles

In a server with complicated role hierarchy, it can be nice for when a user is muted to remove all of their roles, this is what it does! And gives them back once they've been unmuted.

Toxicity Filter

Using advanced AI, we scan every message for anything remotely toxic, this can be very strict but great for only-positive communities.

OCR (Optical Character Recognition)

Using advanced AI, the bot will scan every image sent for text, and then match that up against the filter (including server filter)! Removing another way users can bypass the filter.

Increased Maximums

A list of increases:

  • Filter: Maximum words from 150 to 1500
  • Uncensor List: Maximum words from 150 to 1500
  • Pop Message: Delete after time from 120 seconds to 600
  • Pop message: Max content size from 200 characters to 1000
  • Ignored Channels: Max amount from 5 to infinite
  • Ignoreed Roles: Max amount from 4 to infinite

Less noteable:

  • DM Command Responses: Makes responses to commands respond in the invokers DM's rather than in the same channel.
  • DM Popup Messages: sends the popup message to a users DMs rather than in the chat.
  • Long lasting HelpME codes, allowing staff to diagnose your problem over the span of months.
